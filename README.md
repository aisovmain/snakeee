🎮 Управление
W ↑ вверх

A ← влево

S ↓ вниз

D → вправо

ESC — выход из игры

🎯 Цель игры
Управляй зелёной змейкой (@ — голова, o — тело).
Ешь красные яблоки "O" — змейка растёт, очки увеличиваются.
Не врезайся в границы (#) и в себя!

📊 Очки
В левом верхнем углу: SCORE: XX
Каждое яблоко = +1 очко.

🚀 Особенности
Поле: 78x22 символа (границы #)

Скорость: 85мс между движениями (динамично!)

Без мерцания: плавная картинка

🏁 Начало и конец
Запусти → крупный титульник "ЗМЕЙКА" (синий)

Нажми любую клавишу

Играй!

Конец: экран "ИГРА ОКОНЧЕНА!" + финальные очки
